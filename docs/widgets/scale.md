# Scale (lv_scale)

## Overview

Scale allows you to have a linear scale with ranges and sections with custom styling.

## Parts and Styles
- `LV_PART_MAIN` Main line and major ticks.
- `LV_PART_ITEMS` Minor ticks.
- `LV_PART_INDICATOR` Labels for major ticks (if enabled).
